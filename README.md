# Агент с искусственным интеллектом

Современный интерактивный чат с искусственным интеллектом и функцией веб-поиска. AI agent имеет понятный интерфейс, и сочетает в себе диалоговый искусственный интеллект с возможностями поиска в реальном времени

## ✨ Функции

- **Ответы ИИ в режиме реального времени** — ответы генерируются по мере поступления
- **Встроенный веб-поиск** — ИИ может искать актуальную информацию в интернете
- **Контекстная память** — сохраняет контекст на протяжении всего разговора
- **Прозрачность процесса поиска** - Визуальные индикаторы показывают этапы поиска, чтения и записи.
- **Адаптивный дизайн** - Чистый современный пользовательский интерфейс, который работает на всех устройствах.

## 🏗️ Архитектура

AI agent имеет клиент-серверную архитектуру:

### Клиент (Next.js + React)
- Современное приложение на React, созданное с помощью Next.js
- Потоковая передача обновлений в реальном времени с помощью событий, отправляемых сервером (SSE)
- Компоненты для отображения сообщений, статуса поиска и обработки ввода
### Сервер (FastAPI + LangGraph)
- Серверная часть на Python с использованием FastAPI для конечных точек API
- Реализация LangGraph для диалогового потока с использованием больших языковых моделей и инструментов
- Интеграция с Tavily Search API для возможностей веб-поиска
- События, отправляемые сервером, для потоковой передачи ответов ИИ в режиме реального времени

## 🚀 Начало работы

### Необходимые условия

- Node.js 18+
- Python 3.10+
- Ключ API ChatGroq
- Ключ API Tavily

### Установка и запуск (локально)

Клонируй репозиторий и перейди в папку проекта:

```bash```
git clone https://github.com/volodyapushinskiy/AI-Agent.git
cd AI-Agent

2. **Настроить сервер**
cd server
   python -m venv venv
   исходный код venv\Scripts\activate
   pip install -r requirements.txt

3. **Настройте переменные среды**  
   Создайте файл `.env` в каталоге сервера.:
   GROQ_API_KEY=ваш_groq_api_key
   TAVILY_API_KEY=ваш_tavily_api_key
   
4. **Настройка клиента**
```bash```
cd ../Client
npm install

### Запуск приложения

1. **Запустите сервер**
  ```bash```
   cd Server
   uvicorn app:app --reload

2. **Запустите клиент**
   ```bash```
   cd Client
   npm run dev (npm.cmd run dev)

3. **Откройте браузер и перейдите по адресу http://localhost:3000**   

### 🔍 Как это работает

1. **Пользователь отправляет сообщение** через интерфейс чата
2. **Сервер обрабатывает сообщение** с помощью ChatGroq
3. **ИИ решает**, использовать ли поиск или ответить напрямую
4. Если нужен поиск:
   - поисковый запрос отправляется в Tavily API
   - результаты обрабатываются и передаются ИИ
   - ИИ использует эту информацию для формирования ответа
5. **Ответ отправляется** клиенту в режиме реального времени
6. **Отображаются этапы поиска** (поиск, чтение, запись)
